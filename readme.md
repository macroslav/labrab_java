## Лабораторная №1 

*Вариант 10*

Картотека налоговой инспекции состоит из записей двух типов: физические и
юридические лица. Физические лица генерируются каждые N1 секунд с вероятностью P1.
Юридические лица генерируются каждые N2 секунд с вероятностью P2.
